# AttackersCombat
A simple smallfont combatlogskript.

# Notes
The base combat is 15 seconds. The worldguard region in which the player is knockedaway from is set to spawn by default, you can change this in line 114. Some blocked commands by default are: /spawn, /warps, /warp (rank 1-3), /home, /afk, /tpa and /enderchest and its aliases. these can be changed in lines 41-87. all sent messages are in small font.
